# AnimaShorts AI 🎌

Générateur de shorts anime style TikTok/YouTube Shorts — 100% gratuit.

Un prompt → un MP4 1080×1920 avec narration IA, visuels anime, effets cinématiques et musique.

## Stack gratuit
- Script : Groq Llama 3.3 70B
- Voix : edge-tts (Microsoft Edge Neural)
- Images : HuggingFace Inference API
- Vidéo : MoviePy + FFmpeg
